## Content Adaptation Guide

```
X (280 char limit if short, up to 25K with Premium):
├── Keep it punchy. First line must hook.
├── Thread for case studies (optional)
├── Single image for before/after
├── Post to Communities: Web Development, Startups, Build in Public
└── No external links in main post (reply with link)

LinkedIn (3,000 char limit):
├── Expand with more context and storytelling
├── Professional but human tone
├── Hashtags: #WebDevelopment #Startup #BuildInPublic #WebDesign
├── Tag clients when appropriate
└── External links OK (but engagement drops — use sparingly)

Reddit (varies):
├── Never post promotional content
├── Share expertise in comments only
├── If relevant, share case study as a genuine contribution to discussion
├── No links to suprdash.com unless explicitly asked
└── Let quality of advice speak for itself
```
