# Bio Optimizer

Optimize social media bios for maximum impact and conversions.

## Skill: `/bio-optimizer`

**CRITICAL: Read `_core-principles.md` first. All content must be human-authentic and undetectable as AI-generated. Never reveal AI identity.**

---

## Human-Authenticity Rules for Bios

### Write Bios That Feel Real
- Sound like a human wrote it, not a generator
- Include personality, not just positioning
- Avoid formulaic structures everyone uses
- Add quirks or specific details
- Make it memorable, not generic

### AVOID AI/Generic Bio Patterns
- "Passionate about [thing]"
- "Helping [audience] achieve [result]"
- "On a mission to [generic goal]"
- Emoji-separator-emoji-separator patterns
- Every line perfectly parallel

### DO Write Bios Like This
- Specific, memorable details
- Personality over positioning
- Actual opinions or quirks
- Natural flow, not bullet points
- Something only THIS person would say

### 5. Keywords for Searchability

**In Name Field:**
Add searchable terms people might look for
- "Sarah | Social Media Coach"
- "John Smith - Web Designer NYC"

**In Bio:**
Include industry terms naturally
- "Helping small businesses grow on Instagram"

### 6. Common Bio Mistakes

**Avoid:**
- Vague descriptions ("Lover of life")
- No clear value proposition
- Missing CTA
- Broken or generic links
- Too many emojis
- Overused phrases ("Passionate about...")
- Not using keywords
- Different bios sending mixed messages
