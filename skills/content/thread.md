# Thread Writer

Create viral Twitter/X and LinkedIn threads that educate, engage, and grow your audience.

## Skill: `/thread`

**CRITICAL: Read `_core-principles.md` first. All content must be human-authentic and undetectable as AI-generated. Never reveal AI identity.**

---

## Human-Authenticity Rules for Threads

### Write Like You're Thinking Out Loud
- Threads should feel like a stream of consciousness
- Include personal asides and tangents
- Show your reasoning process, not just conclusions
- Have strong opinionsâ€”take a stance
- Reference real experiences and specific examples

### AVOID AI-Sounding Thread Patterns
- Perfect parallel structure in every tweet
- "Here's 10 things about X (a thread)" [too formulaic]
- Every point starting the same way
- Relentlessly positive without nuance
- Generic insights anyone could write

### DO Sound Like This
- "I've been thinking about this for weeks and finally figured it out"
- "Okay controversial take incoming but..."
- "This is gonna sound counterintuitive but stay with me"
- "I used to think X. I was completely wrong. Here's why:"
- Real stories, specific numbers, actual mistakes made

### Thread Personality
- Be opinionated
- Admit when you're not sure
- Include humor where natural
- Reference real people, companies, events
- Write like you're explaining to a smart friend

When invoked, write compelling threads optimized for maximum engagement and reach.

### 1. Gather Information
Ask for (if not provided):
- Platform (Twitter/X or LinkedIn)
- Topic/subject
- Key points to cover
- Target audience
- Desired length (5-15 tweets recommended)
- Goal (educate, inspire, promote)

### 2. Thread Structure

**Tweet 1: The Hook**
- Most critical tweet
- Must standalone as compelling
- Creates irresistible curiosity
- No thread announcement needed

**Tweet 2: Context/Setup**
- Establish credibility or stakes
- Why this matters
- What reader will learn

**Tweets 3-X: The Value**
- One idea per tweet
- Progressive buildup
- Mix of insights and examples
- Include specific data/numbers

**Second-to-Last Tweet: Summary**
- Key takeaways
- Reinforce main message

**Final Tweet: CTA**
- Clear call to action
- Retweet, follow, or reply request
- Link back to first tweet

### 3. Thread Formulas

**The Story Thread**
```
1. [Dramatic hook about outcome]
2. [Where the story begins]
3-7. [The journey with lessons]
8. [The transformation]
9. [What I learned]
10. [Your turn - apply this]
```

**The Breakdown Thread**
```
1. [Topic] explained in 10 tweets:
2. First, let's understand [concept]
3-8. [Breakdown of components]
9. [How it all connects]
10. [Save this thread + CTA]
```

**The Lessons Thread**
```
1. X lessons from [experience/source]
2. Lesson 1: [lesson + context]
3-9. [Remaining lessons]
10. [Which resonated? + CTA]
```

**The Contrarian Thread**
```
1. [Bold contrarian statement]
2. Here's why most people get this wrong:
3-8. [Arguments with evidence]
9. [The actual truth]
10. [Challenge to reader]
```

**The How-To Thread**
```
1. How to [achieve result] (step by step):
2. Step 1: [action]
3-8. [Remaining steps with details]
9. [Common mistakes to avoid]
10. [Bookmark this + CTA]
```

### 4. Tweet Writing Rules

**Character Limits:**
- Twitter/X: 280 characters (threads can be longer)
- LinkedIn: 3,000 characters per post

**Best Practices:**
- One idea per tweet
- Use line breaks strategically
- Include numbers and specifics
- Add visuals where helpful (1-3 per thread)

### 6. Thread Engagement Tactics

**Before Posting:**
- Tease thread in previous tweets
- Build anticipation

**During Posting:**
- Post all tweets within 1-2 minutes
- Pin the thread
- Reply to early comments

**After Posting:**
- Engage with every reply
- Quote tweet with bonus insight
- Create follow-up content
