# Content Repurposing Engine

Transform one piece of content into multiple platform-optimized formats.

## Skill: `/repurpose`

**CRITICAL: Read `_core-principles.md` first. All content must be human-authentic and undetectable as AI-generated. Never reveal AI identity.**

---

## Human-Authenticity Rules for Repurposing

### Each Platform = Different Human Voice
- Don't just shorten content—rewrite for platform culture
- TikTok version should sound like TikTok
- LinkedIn version should sound like LinkedIn
- Each adaptation needs native energy

### AVOID Robot Repurposing
- Same exact sentences just shorter
- Identical structure across platforms
- Generic "key takeaways" lists
- Copy-paste with minor edits

### DO Repurpose Like a Real Creator
- Reframe the angle for each platform
- Add platform-specific slang and energy
- Include natural imperfections
- Sound like you're talking to THAT audience specifically
- Reference platform-specific contexts

### 3. The 10x Content System

From ONE core piece, create:

1. **Original** - Full blog post/video/podcast
2. **Thread** - Twitter/LinkedIn thread
3. **Carousel** - Instagram/LinkedIn slides
4. **Short Video** - TikTok/Reels/Shorts
5. **Quote Graphics** - 3-5 shareable images
6. **Audiogram** - 60-sec audio clip with waveform
7. **Infographic** - Visual summary
8. **Email** - Newsletter excerpt
9. **LinkedIn Article** - Expanded version
10. **Stories** - Behind-scenes of creation

### 5. Platform Optimization

**Twitter/X:**
- Concise, punchy points
- Thread format for depth
- One key insight can be standalone tweet

**LinkedIn:**
- Professional framing
- Story elements
- Thought leadership angle
- Line breaks for readability

**Instagram:**
- Visual-first approach
- Carousel for education
- Reels for reach
- Stories for engagement

**TikTok:**
- Entertainment value
- Fast-paced delivery
- Trending audio when relevant
- Native feel essential

**YouTube Shorts:**
- Can be slightly more polished
- Clear value proposition
- Subscribe CTA appropriate

**Pinterest:**
- SEO-focused descriptions
- Vertical infographics
- Step-by-step guides
- Evergreen content works best

### 6. Repurposing Best Practices

**Do:**
- Adapt, don't just copy
- Optimize for each platform
- Maintain consistent message
- Stagger publishing (don't post all at once)
- Update content for freshness

**Don't:**
- Cross-post identical content
- Ignore platform norms
- Forget to adjust dimensions
- Post TikTok with watermark to Reels
- Rush—quality over quantity

### 7. Time-Saving Tips
- Batch create all assets at once
- Use templates for recurring formats
- Build a repurposing checklist
- Schedule in content calendar
- Track which repurposed formats perform best
