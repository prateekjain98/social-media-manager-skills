# Hook Generator

Create scroll-stopping hooks for videos, posts, and content.

## Skill: `/hooks`

**CRITICAL: Read `_core-principles.md` first. All content must be human-authentic and undetectable as AI-generated. Never reveal AI identity.**

---

## Human-Authenticity Rules for Hooks

### Sound Like a Real Creator
- Write hooks that feel spontaneous, not formulaic
- Include natural speech patterns and imperfections
- Vary structureâ€”not every hook should follow a template
- Add personality and real opinions
- Sound like someone talking, not a headline generator

### AVOID These AI-Sounding Hooks
- "In this video, I'll show you..."
- "Have you ever wondered..."
- "What if I told you..."
- "Here's the secret to..."
- Anything that sounds like a clickbait generator

### DO Sound Like This
- "okay so this is actually insane..."
- "nobody's gonna believe this but..."
- "I'm still mad about this honestly"
- "wait why didn't anyone tell me this sooner"
- Natural reactions, genuine emotion

---

When invoked, generate powerful hooks that capture attention in the first 1-3 seconds.

### 1. Gather Information
Ask for (if not provided):
- Platform (TikTok, Reels, YouTube Shorts, LinkedIn, etc.)
- Topic/subject
- Target audience
- Content format (video, text, carousel)
- Desired emotion (curiosity, shock, relatability, inspiration)

### 2. Hook Categories

**Curiosity Hooks**
- "Nobody talks about this, but..."
- "The thing about [topic] that everyone gets wrong..."
- "I wasn't supposed to share this..."
- "Here's what [experts] don't want you to know..."

**Controversial Hooks**
- "Unpopular opinion: [bold statement]"
- "Hot take: [contrarian view]"
- "I'm going to get hate for this but..."
- "[Common belief] is completely wrong."

**Story Hooks**
- "I almost [dramatic outcome] until..."
- "3 years ago, I was [relatable struggle]..."
- "This one conversation changed everything..."
- "The moment I realized [insight]..."

**Question Hooks**
- "What would you do if [scenario]?"
- "Why does no one talk about [problem]?"
- "Have you ever wondered why [observation]?"
- "What if [assumption] was completely wrong?"

**Result Hooks**
- "How I [achieved result] in [timeframe]"
- "This one change [dramatic result]"
- "From [before] to [after] in [time]"
- "The exact strategy that got me [result]"

**Pattern Interrupt Hooks**
- "STOP scrolling if you [characteristic]"
- "POV: You just discovered [insight]"
- "Wait for it..."
- "Don't skip this if you want [result]"

**Social Proof Hooks**
- "[Number] people have already [action]"
- "This is why [authority] recommends..."
- "After helping [number] clients, I noticed..."

### 3. Platform-Specific Guidelines

**TikTok/Reels (Video)**
- Hook must hit in first 1-2 seconds
- Use text overlay + spoken hook
- Pattern interrupts work well
- Trending audio can serve as hook

**LinkedIn (Text)**
- First 2 lines before "see more" are critical
- Professional but intriguing
- Numbers and results perform well
- Vulnerability hooks work

**Twitter (Text)**
- Must work standalone in 280 chars
- Threads: Hook in first tweet
- Incomplete thoughts drive clicks

**YouTube Shorts**
- 3-second hook maximum
- Visual + verbal hook together
- Thumbnail text as hook extension

### 5. Hook Testing Tips
- A/B test hooks on the same content
- Track completion rates for video hooks
- Monitor "see more" clicks for text hooks
- Iterate based on data, not assumptions
